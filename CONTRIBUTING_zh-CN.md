# 贡献指南

感谢你对 ScriptBridge-Fabric 感兴趣！我们欢迎任何形式的贡献。

## 如何参与

### 报告 Bug

如果你发现了 Bug，请在我们的 Issue 追踪器中提交一个新的 Issue。请确保包含以下信息：
- 清晰的标题和描述。
- 复现 Bug 的步骤。
- 你使用的模组版本、Fabric Loader 版本和 Minecraft 版本。
- 任何相关的日志或截图。

### 建议新功能

我们需要你的创意！请提交 Issue 来建议新功能或改进。

### 提交代码 (Pull Requests)

1. Fork 本仓库。
2. 为你的功能或修复创建一个新分支：`git checkout -b my-feature-branch`
3. 提交你的更改：`git commit -m "Add some feature"`
4. 推送到你的分支：`git push origin my-feature-branch`
5. 提交 Pull Request。

请确保你的代码遵循现有的代码风格和规范。

## 开发环境设置

1. 克隆仓库。
2. 运行 `./gradlew genSources` 以生成 Minecraft 源码。
3. 将项目导入你的 IDE（推荐使用 IntelliJ IDEA）。
